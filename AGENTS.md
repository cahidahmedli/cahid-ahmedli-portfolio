# Portfolio project guidance

- Preserve the minimal, modern visual direction and restrained motion system.
- Keep Azerbaijani and English content in sync for every user-facing change.
- Preserve both light and dark themes.
- Keep all project cards linked to their detail pages.
- Do not add Masazir Modern City to the portfolio.
- Reuse the existing images under `public/assets/projects`.
- Before finishing a change, run `npm run build` and verify that all local HTML references resolve.
