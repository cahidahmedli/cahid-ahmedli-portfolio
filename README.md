# Cahid Ahmedli Portfolio

Minimal, bilingual portfolio website with 28 linked project case studies.

## Open in Codex

1. Extract the project ZIP.
2. In the ChatGPT desktop app, open **Codex**.
3. Choose **Open folder** and select the extracted `Cahid-Ahmedli-Portfolio-Codex` folder.
4. Start a new Codex chat for this folder.

## Run locally

```bash
npm install
npm run dev
```

## Included

- Azerbaijani and English language switch
- Light and dark themes
- Responsive navigation and project filtering
- 28 detailed project pages
- Existing Sites project identity in `.openai/hosting.json`

The current published version is available at:
https://cahid-ahmedli-portfolio.cahid-ahmedli.chatgpt.site
